# EduTutor Kenya Platform

A comprehensive tutoring platform connecting students with qualified tutors across Kenya.

## Features
- Student and tutor registration
- Booking and scheduling system
- Secure payment processing (M-Pesa, Stripe)
- Admin dashboard with analytics
- Real-time messaging
- Review and rating system
- Email and SMS notifications
